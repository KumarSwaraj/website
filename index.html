Axe-Throw

Supported Platforms:

MacOS & Windows PC

Skills:

C# | Unity3D | Mixamo | VS

For this project, my goal was to recreate the Axe Throw mechanic from God of War (2018) using Unity. I broke down the steps that I would need to take and they are as follows:Find animations for the character motions,Implement throwing the Axe forward,Implement the Axe collision to stick to walls,Implement the Axe return with a small curve

Once all my character animations were setup I could work on implementing the axe throw. The first thing I did was add a rigidbody component to the axe and set the IsKinematic property to true so that it isnâ€™t effected by gravity. In code, all I needed to do was detached the axe from the player, set its IsKinematic property to false and add a force to the rigidbody.

To get the throw and the animation is so happen at the same time, I went into the animation and created an animation event at the frame I wanted to the throw to happen. Then I call the AxeThrow() function from my script on the animation event. To get the axe to rotate, I created a script on the axe that changes its Euler Angles on update.On that same script, I added Unityâ€™s OnCollisionEnter() function to detect whenever the axe collides with something and just set the axeâ€™s rigidbody IsKinematic property to true to get the axe to freeze.

To get the axe to return, I used the Lerp function and a quadratic bezier curve to make the axe do an arc when returning. By creating an empty game object and making it a child of my character I could use it as a middle curve point for the quadratic bezier formula.For when the axe is returning I added a Pull animation for the character. Finally for polish I added some basic particular effects to the axe. One for the glow of the axe which uses the mesh as a shape for its emission and another single emission of a bunch of particles for when the axe is caught. I also added a trail renderer as well and to add some more impact to the catch I added a screen shake to the camera.

Source code is available on my github account.
